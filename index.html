Usage

Install with windows installer link

Configure OpenSubtitle user with a file in C:\Users[USER]\AppData\Roaming\AutoSub\default.yaml :

OpenSub:
  user: <USER>
  password: <PASSWORD>

Rigth click on a video and "Search with AutoSub"
